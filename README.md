# Phishing Awareness Simulator

## Project Overview

The Phishing Awareness Simulator is a cybersecurity awareness project designed to educate users about phishing attacks, social engineering tactics, and online safety practices. The simulator demonstrates common phishing scenarios and teaches users how to identify and avoid malicious emails, fake websites, and fraudulent communications.

---

## Objectives

- Increase awareness about phishing attacks.
- Demonstrate common phishing techniques.
- Educate users on identifying suspicious emails and websites.
- Promote safe online behavior.
- Reduce the risk of credential theft and data breaches.

---

## Features

- Simulated phishing email examples.
- Common phishing indicators.
- Fake login page awareness.
- Security recommendations.
- Educational cybersecurity content.

---

## Simulation Scenarios

### 1. Banking Phishing Email
A fake email claiming unusual activity on a bank account.

### 2. Password Reset Scam
A fraudulent email requesting immediate password changes.

### 3. Prize Winner Scam
A message informing the user they have won a reward.

### 4. Fake Login Page
A cloned website attempting to steal credentials.

### 5. Business Email Compromise
An impersonation attack targeting employees.

---

## Common Phishing Indicators

- Suspicious sender addresses.
- Urgent requests for action.
- Grammar and spelling mistakes.
- Unexpected attachments.
- Requests for sensitive information.
- Mismatched URLs.

---

## Prevention Techniques

- Verify sender identities.
- Enable Multi-Factor Authentication (MFA).
- Use strong passwords.
- Never click unknown links.
- Inspect URLs carefully.
- Report suspicious communications.
